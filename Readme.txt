Calculator -  консольный калькулятор, который принимает входную строку содержащую математическое выражение (+, -, *, /, скобки) и выводит в консоль его результат. Решение можно расширять другими операциями   
Автор: Медведев Александр Анатольевич    

Base и Trigonometric - проекты для создания библиотек с наборами математических функций. Что бы функции были доступны, необходимо их зарегистрировать в файле operations.xml, который находится в каталоге с исполняемым файлом.

Calculator – проект решения задачи
Calculator\BusinessLogic - Описывает интерфейсы и их взаимодействие
Calculator\Domain - Реализация интерфейсов
Calculator\Tests - Набор тестов
Calculator\Calculator - Готовое приложение
Calculator\Complete application - Скомпилированное и готовое для демонстрации приложение
